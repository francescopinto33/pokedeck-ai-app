# PokeDeck AI

PokeDeck AI ist ein einfaches Anfängerprojekt fuer eine Pokemon-TCG-Deck-App.

## MVP von Version 1

Diese Version kann bereits:

- Karten aus lokalen Testdaten anzeigen
- Decks erstellen
- Karten zu Decks hinzufuegen
- Decks grundlegend pruefen
- Decks im Browser speichern
- eine einfache Sammlung verwalten und speichern

## Was Version 1 noch nicht kann

Diese Version enthaelt bewusst noch nicht:

- perfekte Decklisten generieren
- Turnierdaten automatisch auswerten
- Bilder von Karten erkennen
- Sammlung per Foto einlesen
- Meta komplett analysieren
- echte API-Anbindung
- Datenbank oder Backend
- Login-System

## Verwendete Technik

- Next.js
- TypeScript
- Tailwind CSS
- localStorage
- lokale Testdaten

## Projekt starten

1. Terminal im Projektordner oeffnen
2. Folgenden Befehl ausfuehren:

```bash
npm install
npm run dev
